# evm

> Electron Version Manager

- [x] unix support
- [x] windows support (experimental)
- [x] evm install
- [x] evm use
- [ ] evm ls

```plain
npm install -g evm
```

```plain
evm install 0.33.6
evm install 1.2.1
evm use 1.2.1
evm use 0.33.6
```
